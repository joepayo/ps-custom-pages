question: Will I get the stimulus payment if I haven’t filed taxes?
id: faq-24
date: 2020-01-11
---
**You must file taxes to receive a stimulus check,** unless you are receiving Social Security benefits. If you don’t normally file taxes (or haven’t filed taxes for 2018 or 2019) you need to file a simple federal income tax return as soon as possible.
