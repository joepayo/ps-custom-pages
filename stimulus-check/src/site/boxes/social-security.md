You can file a federal tax return for free online:
* If you receive Social Security benefits and have dependent(s), you should file a tax return to ensure you receive a $500 credit (the Social Security Administration does not have information on dependents).
* If you receive SSI, but not Social Security, you will need to file a tax return to receive your stimulus payment.
